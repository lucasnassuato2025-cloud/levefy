# LEVEFY

Premium weight loss & healthy lifestyle SaaS — MVP.

## Stack
Next.js 15 · TypeScript · Tailwind CSS · App Router

## Install
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Pages
- `/` Landing
- `/login` Auth
- `/dashboard` Main app
- `/recipes` Healthy recipes
- `/challenge` 21-day challenge
- `/profile` User profile
- `/membership` Plans
- `/admin` Admin panel

Mock data only. Connect your backend (Supabase, etc.) when ready.
